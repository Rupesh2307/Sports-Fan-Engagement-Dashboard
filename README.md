# Sports Fan Engagement Dashboard (Tableau)

## Overview
Interactive Tableau dashboard analysing sports fan engagement using social media data (FIFA + NBA). The dashboard explores how engagement varies by sentiment, content type, time trends, and engagement concentration.

## Key Visualisations
- Daily Engagement Trend (time-series)
- Fan Sentiment Distribution (positive/neutral/negative)
- Engagement by Content Type (Behind-the-Scenes, Highlights, Live, Other)
- Engagement Concentration (distribution skew / high-impact posts)

## Interactivity
Selecting a sentiment segment or content type filters all visuals to support exploratory analysis.

## Tools
- Tableau Public (Dashboard + interactivity)
- Excel (initial EDA summaries)
